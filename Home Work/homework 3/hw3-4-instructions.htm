Homework 3-4: Study HTML DOM Events

This page listed many HTML DOM events. Create a web page named hw3-4.htm that applies one or multiple events. Your code should:

  • NOT only use event(s) that we have used in past classes, such as onclick, onmouseover, onmouseout, onload and onchange.
    • In other words, you should study some new events.
  • Be simple to explain the usage of the event(s) you select.
  • Be interesting. It should solve a real-world problem.
  • NOT use same example that you find from the w3schools.com or any other website.
  • Add comments to describe the event and how your code works.
