Homework 3-2: Make your own sentence

Write a function that displays the sentence (consisting of 3 words) that the user creates by selecting one word from each of the three drop-down menus. The sentence must be displayed only when the user picks the third word. No additional words are required. Validate the function by ensuring that the user selects one valid word from each - if not, provide an alert to the user indicating that the user must select a valid word. Nothing should be displayed if the user has not picked one from each list of words.

When the button is clicked, generate a sentence by randomly choosing words from each of the three drop-down lists.
